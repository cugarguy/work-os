# WorkOS Quick Start

## One-Time Setup

```bash
# Install global alias (optional but recommended)
./scripts/install-alias.sh
```

## Daily Usage

### Option 1: From anywhere (if alias installed)
```bash
pos
```

### Option 2: From this directory
```bash
./scripts/ttgo.sh
```

## What happens when you start:
1. ✅ Dependencies installed
2. ✅ MCP server configured  
3. ✅ Directories created
4. ✅ Ready for Q chat

## Then use Q chat:
```bash
q chat
```

## Common commands in Q chat:
- `"process my backlog"` - Turn BACKLOG.md into organized tasks
- `"what should I work on?"` - Get AI suggestions
- `"show me P0 tasks"` - See urgent items
- `"mark task X as done"` - Complete work

## Files:
- **BACKLOG.md** - Brain dump here (no structure needed)
- **GOALS.md** - Your priorities and vision
- **Tasks/** - Organized task files
- **Knowledge/** - Reference docs and notes
