# People Tracker

Track conversations, relationships, and context with people you interact with regularly.

## How to Use

1. **Create a new person file**: Copy `core/templates/person.md` and rename it to `[FirstName LastName].md`
2. **Fill in contact info** and relationship context
3. **Add meeting notes** after each interaction
4. **Link to related tasks** in the Tasks/ folder
5. **Update regularly** to maintain context

## File Naming Convention
- Use format: `[FirstName LastName].md`
- Examples: `John Smith.md`, `Sarah Chen.md`

## Quick Commands for AI Assistant
- "Create a new person file for [Name]"
- "Add meeting notes for [Name] from today"
- "Show me recent interactions with [Name]"
- "Link this task to [Name]'s file"

## Integration with Tasks
- Reference people in task files using `People/[Name].md`
- Link tasks to people for better context tracking
- Use people files to understand stakeholder relationships

## Privacy Note
These files contain personal interaction data. Keep them secure and respect privacy.
